# ASCII Art Frontend

This project is a React frontend for the ASCII Art API.

It allows users to upload an image and convert it into ASCII art through the backend service.

## Features

* Upload images from the browser
* Convert images to ASCII art
* Display ASCII output in real time
* Simple and responsive user interface

## Requirements

* Node.js
* npm

## Installation

Clone the repository:

```bash
git clone https://github.com/and-pad/asciiart_frontend.git
cd asciiart_frontend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

## Backend

This frontend requires the ASCII Art Django API to be running.
